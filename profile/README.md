An applied-AI workforce programme for diploma engineers across every branch. Students build, deploy, document, and defend real systems. This organisation hosts the working portfolios of each cohort. Every repository is a defendable artefact, not a notebook dump.

### How We Work

**🔁 Learning Loop**
Concept → Demo → Guided Lab → Practice → Reflection

**📝 Evidence of Work** — every project repository must contain:
* `README.md`
* `AI_USAGE.md`
* `prompt-log.md`
* Module deliverables
* Supporting documentation and assets

**⚖️ Assessment Standard**
* AI use must be disclosed.
* Prompts must be documented.
* Outputs must be verified.
* Students must be able to explain and defend their work.

**🛠️ Technology Stack**
* Google Colab
* GitHub
* Hugging Face
* Python
* scikit-learn
* TensorFlow / Keras
* FastAPI
* Edge Impulse
* No-Code AI Platforms

### Student Portfolio Setup

Every student is required to maintain an individual GitHub portfolio repository throughout the programme.

**Step 1** — Visit the Foundation Template repository:
https://github.com/ooumph-coe-bbsr/foundation-template

**Step 2** — Click **Use this template → Create a new repository**

**Step 3** — Create your repository using the naming convention:

```text
f26-<roll>-<firstname>
```

Examples:

```text
f26-019-rahul
f26-022-priya
```

**Step 4** — Create the repository under your own GitHub account.

**Step 5** — Submit your repository URL to the trainers during onboarding.

**Step 6** — Make your first commit before leaving the setup lab.

### For Trainers

Trainers are responsible for:
* Reviewing repository activity

Repository activity should be monitored regularly to identify students requiring additional support.

### Governance & Ethics

Operated under the Government Polytechnic Bhubaneswar × MyOoumph Networks Applied AI CoE framework.

**Programme Principles:**
* Responsible use of AI
* Transparent disclosure of AI assistance
* Evidence-based assessment
* Practical, project-driven learning
* Respect for privacy and consent

**No facial-recognition deployments.** All project work involving image, video, or personal data must be supported by appropriate consent and privacy safeguards.
